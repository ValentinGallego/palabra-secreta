# Palabra secreta

 Juego social de palabras en el que todos los jugadores reciben una palabra secreta, excepto uno que cumple el rol de impostor. El objetivo es que los jugadores encuentren al impostor antes de que el mismo descubra la palabra.

 ## Características
- Partidas multijugador (pensado para jugar con amigos).
- Roles: jugadores normales e impostor.
- Múltiples categorias de palabras.
- Rondas de preguntas y de votación.
- Flujo de sala (host / jugadores) y control de partida.
- Aplicación mobile para Android.

 ## Cómo jugar
 - Descargá la apk desde [acá](https://github.com/ValentinGallego/palabra-secreta/releases/tag/v0.1.0) en tu dispositivo Android.
 - Creá una sala (puede demorar unos segundos mientras se levanta el servidor).
 - Otros jugadores pueden unirse desde la aplicación.
 - El host selecciona las categorías y comienza la partida.
 - Durante la ronda, los jugadores hacen preguntas o dan pistas sin revelar la palabra secreta.
 - Se realiza una votación para intentar identificar al impostor.

 ## Notas técnicas

- Juego desarrollado como proyecto personal.
- Arquitectura cliente-servidor.
- Comunicación entre jugadores mediante servidor central.
- Enfoque en lógica de juego, manejo de estados y flujo de partida.
